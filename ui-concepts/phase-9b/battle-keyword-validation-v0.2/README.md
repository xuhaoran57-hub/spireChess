# 旅团绘本战斗关键词书签与护盾覆盖高保真验证 v0.2

## 状态

- v0.1 降级为结构线框，不作为视觉评审依据。
- v0.2 使用正式战斗界面截图、真实 160×240 卡牌和旅团绘本材质语言。

## 关键词书签

- 嘲讽：陶土红粗布盾旗，黄铜/米白缝线，最高显示优先级。
- 亡语：紫灰纸蜡封，铃铛与裂纹，死亡结算期间点亮。
- 溅射：青绿皮革分叉书签，攻击预告时展开目标支线。
- `+N`：黄褐目录页签；四个及以上关键词时显示两个最高优先级关键词和溢出数量。
- 战吼不常驻；永久成长不创建关键词书签。

## 护盾覆盖

- 完整：双层冷蓝水彩边缘，独立于关键词栏。
- 抵挡：受击点产生蓝白水彩飞溅，生命不变。
- 破裂：从受击点扩散裂纹，水彩边缘变为断续。
- 消失：边缘完全移除，只保留向外飞散的短暂碎片。

## 多关键词规则

1. 单关键词完整显示。
2. 双关键词纵向排列，不重叠主体。
3. 三关键词为常驻上限。
4. 护盾与关键词栏可叠加，互不占用空间。
5. 四个及以上显示前两个关键词和 `+N`，详情面板展示全部。
