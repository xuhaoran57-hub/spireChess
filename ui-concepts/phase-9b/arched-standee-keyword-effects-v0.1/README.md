# 拱顶旅团肖像立牌关键词效果 v0.1

本稿使用内置 GPT Image 工作流生成。五个玩家单位从左到右依次为：

1. 无关键词基准。
2. 护盾。
3. 嘲讽。
4. 亡语。
5. 溅射。

## 空间与材质

- 护盾：肖像表面与拱顶轮廓，冰蓝半透明水彩壳。
- 嘲讽：立牌外侧与底座，陶红旅团盾肩、织物与黄铜。
- 亡语：立牌背后，紫灰裂蜡、残纸与微弱灵性颜料。
- 溅射：武器前方的攻击关系层，青绿分叉水彩与两个金色命中点。

四种效果刻意使用不同颜色、材质、空间层和外轮廓，避免退化为四种发光描边。

## 约束

- 不显示名称、说明、等级或种族。
- 攻击和生命保持无遮挡。
- 本图验证的是静默识别态，不是完整触发动画。
- 后续需要单独验证双关键词、三关键词和四关键词同时存在时的降亮度规则。
