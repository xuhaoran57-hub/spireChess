## 费用

```text
Create one isolated game-UI sprite for a Chinese wandering-storybook fantasy
card game.

Subject: a very small, lightweight COST marker: one flat front-facing
aged-brass coin/tag with a thin irregular rim, restrained hammered texture,
and a tiny simple leather fastening loop. It must have a clean empty circular
center where a runtime number will be overlaid later.

Match the project's warm handmade paper, ink, indigo stitching, worn leather,
and restrained antique-metal visual language. Quiet, practical, low contrast;
fine ink-like edge; readable at 20–30 px.

Exactly one centered orthographic object on perfectly uniform solid chroma
green #00FF00. No number, letters, symbol, emblem, laurel, crown, gems, rays,
flourish, thick bevel, dramatic relief, cast shadow, glow, card frame, mockup
or extra objects. Do not use green on the object.
```

## 等级

```text
Create one isolated game-UI sprite for a Chinese wandering-storybook fantasy
card game.

Subject: a very small, narrow TIER marker shaped like a slim five-sided
bookmark tab. Neutral warm flax/ochre cloth or paper-leather, a hairline ink
contour, a few tiny quiet stitches near the edge, and a clean empty center for
a runtime number. It must remain readable at 16x20 and 24x30 pixels.

Flat, understated, low-contrast, practical. Exactly one centered orthographic
object on perfectly uniform solid chroma green #00FF00. No number, letters,
icon, emblem, heraldry, crown, gems, rays, flourish, tassel, thick border,
deep folds, relief, shadow, glow, card frame, mockup or extra objects. Do not
make a large pennant, shield or badge. Do not use green on the object.
```

## 攻击

```text
Create one isolated game-UI sprite for a Chinese wandering-storybook fantasy
card game.

Subject: one very low-profile horizontal ATTACK number tag. On the LEFT is a
tiny fixed angular cap of blackened iron with an extremely subtle short-sword
relief; extending to the RIGHT is a short, low-height blank dark-paper/iron
capsule that can visually stretch inward for one to four runtime digits. Keep
a clean empty center. Thin copper-ink edge only.

Flat silhouette, quiet values, crisp at 18–26 px high. Exactly one centered
orthographic object on perfectly uniform solid chroma green #00FF00. No
number, letters, large sword, crossed weapons, shield, spikes, large round
medallion, ornament, thick bevel, relief, shadow, glow, card frame, mockup or
extra objects. It must feel embedded in the card's bottom edge. Do not use
green on the object.
```
