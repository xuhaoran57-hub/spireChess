# 拱顶旅团肖像立牌关键词组合验证 v0.2

本稿使用内置 GPT Image 工作流生成。五个玩家单位从左到右依次为：

1. 新版亡语。
2. 新版溅射。
3. 嘲讽＋亡语。
4. 护盾＋亡语＋溅射。
5. 护盾＋嘲讽＋亡语＋溅射。

## 修订

- 亡语改为拱顶正上方的紫灰裂蜡封，不再占用立牌侧边和底座。
- 溅射静默态改为攻击圆章上方的小型青绿三叉刃标记，不显示目标线或命中动画。
- 嘲讽继续使用立牌外侧的陶红盾肩和底部挑战毯。
- 护盾继续使用肖像表面的冰蓝半透明水彩壳。

## 固定区域

- 顶部：亡语。
- 表面：护盾。
- 外侧与底部：嘲讽。
- 攻击圆章上方：溅射。

静默态只负责识别。溅射目标连线、亡语蜡封开裂、护盾破碎和嘲讽目标牵引均留到对应事件动画中表现。
