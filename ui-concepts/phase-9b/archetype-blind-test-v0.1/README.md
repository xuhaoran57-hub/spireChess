# Phase 9B 三流派与旅团 5 秒盲测卡 v0.1

日期：2026-07-24

## 使用方式

1. 只向参与者展示 `cards/` 中的 `A.png` 至 `H.png`，不要展示 `masters/` 或本文件后面的答案。
2. 每张展示 5 秒，不提供卡名、流派名称、文字、图标、边框、数值或技能说明。
3. 每张只能选择：`铸魂`、`荒灵`、`星契`、`旅团`。
4. 记录第一判断、信心分（1–5）和误判原因；不要在中途公布答案。
5. 建议随机化 A–H 的展示顺序，并至少邀请 5 名未参与美术制作的人独立测试。

## 通过建议

- 单卡正确率至少 80%。
- 每个类别的两张卡合计正确率至少 80%。
- 旅团误判为任一主流派的比例不超过 20%。
- 若不通过，只调整 Style Tile 的轮廓、主色或形状语言，不使用文字、图标或边框补救。
