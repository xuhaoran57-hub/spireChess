## 结论

立牌主体只常驻肖像、攻击和生命；普通立牌使用银灰/白镴框，金色立牌保持暖金。嘲讽位于底座、亡语位于拱顶、溅射依附攻击圆章、护盾覆盖肖像。合法目标为细绿空心边线，不使用整块青色填充；护盾 Additive Alpha 提升至 0.78 并扩大至 132×222。槽位和立牌根节点背景全透明但保留拖放 Raycast，因此战斗区只显示空槽编号、拱顶立牌与必要状态。Hover 临时显示完整卡牌，点击锁定/解除详情。没有专属插画时按流派色显示首字回退，不增加单卡代码分支。

锁定详情复用的 `PF_Card` 已切换为旅团绘本普通/金色公共框：暖纸、深墨、靛蓝缝线与局部金箔取代黑银厚金属框。完整卡牌和商店双分辨率证据分别见 `../pf-card-v0.1/` 与 `../pf-shop-screen-v0.1/`。

本验证只证明 G1 Unity 接线与可读性，不代表生成式资产获得最终发布许可，也不替代 G1 的共同风格、流派区分、金色表现和批量生产成本人工评审。
